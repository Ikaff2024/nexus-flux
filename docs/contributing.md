# Contributing

Merci de votre intérêt pour contribuer !

---

## Comment contribuer ?

- ouvrir une issue,
- suggérer une amélioration,
- ajouter un agent cognitif,
- améliorer le state backend,
- améliorer la doc.

---

## Processus

1. Fork  
2. Nouvelle branche  
3. Commit clair  
4. Pull Request

---

## Qualité attendue

- code lisible,
- commentaires utiles,
- documentation mise à jour,
- tests ajoutés si nécessaire.
