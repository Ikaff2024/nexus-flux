# Roadmap

## Court terme (1–2 semaines)
- version stable du moteur safe,
- documentation complète,
- tests unitaires,
- exemples supplémentaires.

---

## Moyen terme (1–2 mois)
- backend Redis stable,
- interface web minimaliste,
- extension du logger,
- nouveaux agents (stabilité, compression, explicabilité).

---

## Long terme
- version distribuée,
- performances + scalabilité,
- nouvelle architecture vectorielle,
- intégration avec moteurs RAG,
- version entreprise.
