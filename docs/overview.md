# NEXUS-FLUX — Overview

NEXUS-FLUX est un moteur d’intelligence collective non anthropomorphique basé sur :

- une communication 100 % vectorielle,
- un état holographique partagé,
- des agents cognitifs spécialisés,
- un mécanisme de convergence du momentum,
- et un consensus vectoriel final.

Cette version est la version **open-source safe**, ne contenant aucune partie brevetable.

---

## Objectifs

- Explorer une alternative aux architectures multi-agents classiques.
- Proposer un modèle où les agents ne s’envoient pas de messages : ils écrivent et lisent dans un espace partagé.
- Fournir un moteur simple mais extensible.
