# Agents cognitifs

Chaque agent représente une stratégie cognitive.

---

## EntropicAgent  
Génère idées, variations, contradictions faibles.

## NegentropicAgent  
Raffine, renforce les pistes prometteuses.

## DoubtAgent  
Introduit scepticisme, diversifie.

## ContextualAgent  
Donne cohérence aux informations.

## PragmaticAgent  
Produit la sortie finale.

---

Les agents héritent de `BaseCognitiveAgent`.
